div dan span

"tag pada HTML digunakan untuk memberikan 'maksud'/'arti' pada sebuah konten [p untuk paragraf,h1 untuk heading utama, dll]

Tag [div] dan [span] tidak memiliki 'arti' apapun, keduanya dingunkan untuk mengolompokkan tag-tag HTML dan memberikan informasi terhadap tag-tag tersebut.

div = display:blok span = display:inline

setiap tag pada HTML berada pada dalam kotak. properti display pada CSS mengatur perilaku dari kotak terse but
